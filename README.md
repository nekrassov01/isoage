## Installation

Install with homebrew

```sh
brew install nekrassov01/tap/isoage
```

Install with go

```sh
go install github.com/nekrassov01/isoage@latest
```

Or download binary from [releases](https://github.com/nekrassov01/isoage/releases)

## Shell completion

Supported Shells are as follows:

- bash
- zsh
- fish
- pwsh

```sh
isoage completion bash|zsh|fish|pwsh

# In the case of bash
source <(isoage completion bash)
```
